变量和简单的数据类型

发表于 2018-09-17 | 分类于 Python笔记

写在前面

该笔记学习的内容均来自于《Python 编程：从入门到实践》，只是作为一个内容总纲，若需详细学习，可去百度网盘下载 ，密码：7rup

字符串拼接

first_name = "ada"
last_name = "lovelace"
full_name = first_name + " " + last_name
print("Hello, " + full_name.title() + "!")

out：
    Hello, Ada Lovelace!

str() 的使用

age = 20
message = "Happy " + age + "rd Birthday!"
print(message)

out：
    TypeError: Can't convert 'int' object to str implicitly        //报错

以上应改为：
message = "Happy " + str(age) + "rd Birthday!"

学习要点

  1. 如何创建描述性变量名以及如何消除名称错误和语法错误；
  2. 字符串是什么，以及如何使用小写、大写和首字母大写方式显示字符串；
  3. 使用空白来显示整洁的输出，以及如何剔除字符串中多余的空白；
  4. 如何使用整数和浮点数；
  5. 使用数值数据时需要注意的意外行为。
  6. 你还学习了如何编写说明性注释，让代码对你和其他人来说更容易理解。最后，你了解了让代码尽可能简单的理念。
