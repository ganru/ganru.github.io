if 语句的使用

cars = ['audi', 'bmw', 'subaru', 'toyota']
for car in cars:
    if car == 'bmw':
        print(car.upper())
    elif car == 'audi':
        print(car.title() + " is very good!")                
    else:
        print(car.title())
out：
    Audi is very good!
    BMW
    Subaru
    Toyota

常用的判断符号

== 判断是否相等
!= 判断是否不等
and 判断两条件是否同时为 True 。（相当于 && ）
or 判断是否有一个条件为 True 。（相当于 ||）
in 判断特定值是否已在列表中
    ex：
        Array = ['gan','GG']
        'gan' in Array
    out：
        True
not in 判断特定值是否不在列表中
