函数专项

写在前面

该笔记学习的内容均来自于《Python 编程：从入门到实践》，只是作为一个内容总纲，若需详细学习，可去百度网盘下载 ，密码：7rup

函数的定义

关键字 def 定义函数。

ex：
    def hello(username):
        """此处写函数注释，建议每次都写，方便日后维护"""
        print(username.title() + ", Welcome to you!")

    hello('ganru')    #要加单引号，不然编译器会检测为未定义对象而不是字符串变量。

out：
    ganru, Welcome to you!

实参与形参

如 hello() 的函数定义，括号里面的变量 username 是形参，只具有形式意义，而值 ‘ganru’ 是实参，具有实际意义。
如何判断：函数定义中的变量为形参，函数调用中的变量为实参。

实参的传递

位置实参

定义：基于实参的顺序将函数调用中的每个实参都关联到函数定义中的一个形参。

ex：
    def student(name, sex):
        .
        .
        .

    student('garu','boy')    #这里的 'garu' 为第一顺序，'boy' 为第二顺序。

关键字实参

定义：传递给函数的名称—值对。无需考虑函数调用中的实参顺序，还清楚地指出了函数调用中各个值的用途。

ex：
    def student(name, sex):
        .
        .
        .

    student(sex = 'boy', name = 'garu')    #顺序反了也没事

注意：使用关键字实参时，务必准确地指定函数定义中的形参名。

默认值

调用函数中给形参提供了实参时，Python 将使用指定的实参值；否则，将使用形参的默认值。
作用：给形参指定默认值后，可在函数调用中省略相应的实参。

ex：
    def student(name, sex = 'boy'):
        .
        .
        .

    student(name = 'garu')    #以后每次调用，如果没有显式提供实参给 sex，sex 都是 'boy'。

注意：如果以后要用默认值，也就是考虑到可能不写值，则形参放最后面（如：sex），以免出现位置实参错误。

等效的函数调用

就是上面的实参调用方式可以混用。

return

ex：
    def add(first,second):
        """二值相加"""
        value = first + second
        return value

    a = input("first value?")    #要将接收到的数据转变为数值，函数才有效
    a = int(a)
    b = input("second value?")
    b = int(b)
    result = add(a,b)
    print("result is " + str(result))

注意：输出时要给 result 打上str()。
否则会报警：can only concatenate str (not “int”) to str

6.5 实参可选

ex：
    def location(country, province, city):
        """个人居住地"""
        my_location = country + ' ' + province + ' ' + city
        return my_location.title()

这种函数或许适用大多数情况，但是直辖市啥整？
可以试试：

def location(country, city, province = ''):    # province 放后面，以免出现位置实参错误
    """个人居住地"""
    if province:    #如果是空字符串，Python 将解读为 False。
        my_location = country + ' ' + province + ' ' + city
    else:
        my_location = country + ' ' + city
    return my_location.title()

新函数就是所说的实参可选。

字典的返回

ex：
    def location(country, province, city):
        """返回一个字典，其中包含一个人的位置所在地"""
        my_location = {
            'country_name': country,
            'province_name': province,
            'city_name': city,
            }
        return my_location

    mylocation = location('China', 'Guangdong', 'Shenzhen')
    print(mylocation)

out：
    {'country_name': 'China', 'province_name': 'Guangdong', 'city_name': 'Shenzhen'}

当然，直辖市我们也要考虑：

def location(country, city, province = ''):    #默认值放后面
    """返回一个字典，其中包含一个人的位置所在地"""
    if province:
        my_location = {
            'country_name': country,
            'province_name': province,
            'city_name': city,
            }
    else:
        my_location = {
            'country_name': country,
            'city_name': city,
            }
    return my_location

mylocation = location(country = 'China', city = 'Beijing')
print(mylocation)

函数与 while

ex：

def location(country, city, province = ''):
    """返回一个字典，其中包含一个人的位置所在地"""
    if province:
        my_location = {
            'country_name': country,
            'province_name': province,
            'city_name': city,
            }
    else:
        my_location = {
            'country_name': country,

            }
    return my_location

users_location = {}
sign = True
while sign:
    print("\nPlease tell me your name and your location:")
    us_name = input("your name:")
    co_name = input("your country:")
    pr_name = input("your province:")
    ci_name = input("your city:")

    user_location = location(country = co_name, province = pr_name, city = ci_name)
    users_location[us_name] = user_location

    symbol = input("Thank you, can you continue?(yes or no)")
    if symbol == 'yes':
        sign = True
        print(users_location)
    else:
        sign = False

print("Thank you!")

小结：字典中的添加不涉及方法 append()。

向函数传递列表、字典

ex：
    def users_language(language):    #实参有要求的，这里默认导入的实参为字典，若有其他需求，可以加个 if 语句控制实现需求
        """展现各个人分别喜欢的语言"""
        for key, value in language.items():
            print("\n" + key + "'s favorite language is " + value)

    favorite_languages = {
            'jen': 'python',
            'sarah': 'c',
            'edward': 'ruby',
            'phil': 'python',
            }

    users_language(favorite_languages)

out：
    jen's favorite language is python

    sarah's favorite language is c

    edward's favorite language is ruby

    phil's favorite language is python

用函数修改列表、字典

ex：
    def print_models(unprinted_designs, completed_models):
        """
        模拟打印每个设计，直到没有未打印的设计为止
        打印每个设计后，都将其移到列表completed_models中
        """
        while unprinted_designs:
            current_design = unprinted_designs.pop()

            # 模拟根据设计制作3D打印模型的过程
            print("Printing model: " + current_design)
            completed_models.append(current_design)

    def show_completed_models(completed_models):
        """显示打印好的所有模型"""
        print("\nThe following models have been printed:")
        for completed_model in completed_models:
            print(completed_model)

    unprinted_designs = ['iphone case', 'robot pendant', 'dodecahedron']
    completed_models = []

    print_models(unprinted_designs, completed_models)
    show_completed_models(completed_models)

out：
    Printing model: dodecahedron
    Printing model: robot pendant
    Printing model: iphone case

    The following models have been printed:
    dodecahedron
    robot pendant
    iphone case

禁止函数修改列表、字典

根据上节的打印可知，unprinted_designs 打印完后列表变为空的了，若想保持其不变，我们可以建立一个列表副本：
print_models(unprinted_designs[:], completed_models) #切片建立副本
但建议在有充分的理由需要传递副本的情况下这么做，毕竟多建一个列表，时间复杂度增加，内存使用率增大，不利于做大项目。

任意数量的实参传递

传递任意数量的实参 （建立空元组）

常见作用：
有时候真不知道函数要接收多少个实参，这时候可以对形参名加个 * ，让 Python 创建一个空元组，并将收到的所有值都封装到这个元组中。

ex：
    def make_pizza(*toppings):
        """概述要制作的比萨"""
        print("\nMaking a pizza with the following toppings:")
        for topping in toppings:
            print("- " + topping)

    make_pizza('pepperoni')
    make_pizza('mushrooms', 'green peppers', 'extra cheese')

函数调用解析：
这时候 * 号调用作用于 tuple/list/str 前，意思是解包(unpack)tuple/list/str，解包后的 tuple/list/str 将不再是一个参数，而是每个元素都是一个函数参数，依次传递给函数。

out：
    Making a pizza with the following toppings:
    - pepperoni
    Making a pizza with the following toppings:
    - mushrooms
    - green peppers
    - extra cheese

结合使用位置实参和任意数量实参

ex：
    def make_pizza(size, *toppings):
        """概述要制作的比萨"""
        print("\nMaking a " + str(size) +
            "-inch pizza with the following toppings:")
        for topping in toppings:
            print("- " + topping)

    make_pizza(16, 'pepperoni')
    make_pizza(12, 'mushrooms', 'green peppers', 'extra cheese')
out：
    Making a 16-inch pizza with the following toppings:
    - pepperoni

    Making a 12-inch pizza with the following toppings:
    - mushrooms
    - green peppers
    - extra cheese

使用任意数量的关键字实参 （建立空字典）

预先不知道传递给函数的会是什么样的信息，有多少个，这时可以用关键字实参调用。即接收任意数量的 键-值 对。
格式：对形参加 ** ，让Python创建一个空字典，并将收到的所有 键-值 对都封装到这个字典中。

ex：
    def build_profile(first, last, **user_info):    #**号创建空字典
        """创建一个字典，其中包含我们知道的有关用户的一切"""
        profile = {}
        profile['first_name'] = first
        profile['last_name'] = last
        for key, value in user_info.items():
            profile[key] = value
        return profile

    user_profile = build_profile('albert', 'einstein',
                                location='princeton',
                                field='physics')
    print(user_profile)
out：
    {'first_name': 'albert', 'last_name': 'einstein', 'location': 'princeton', 'field': 'physics'}

函数与模块

导入整个模块

模块是后缀 .py 的子文件，在模块中写好函数（除函数外，其余都不要写），然后在主文件中用 import 导入整个模块。
ex：

out：

jen's favorite language is python

sarah's favorite language is c

edward's favorite language is ruby

phil's favorite language is python

导入特定函数及给函数指定别名

格式：
from module_name（.py 文件名） import function_name（在 .py 文件中的函数名）
导入多个函数：
from module_name import function_0, function_1, ...
若函数名太长或与程序现有名称冲突：（指定别名区分）
from module_name import function_0 as f_0

给模块指定别名

格式：
import son as s #比如上述中的 son.py 函数文件

使用的时候，son.users_language(favorite_languages) 可以这么写：

def print_models(
        unprinted_designs, completed_models)
```

输入左括号后换行，并输入两次 Tab ，便于观看。

  1. 如果模块很多函数，可以在两个函数之间换两次行。
  2. 所有的import语句都应放在文件开头，唯一例外的情形是，在文件开头使用了注释来描述整个程序。
