2.7 列表中 for 的使用
跟 C、C++ 一样，用 for 语句循环遍历，不过 python 的使用方式有所不同。
如 for cat in cats:
.
. //循环语句块（带上缩进才算，即写完 for 换行之后，缩进该行）
.
表示从列表 cats 选取第一个元素并赋予 cat，然后执行语句块，直至选取到列表 cats 中最后一个元素。

注意这里的命名规则，尽量用复数名词表示列表（如 cats），用单数名词表示单个列表元素（如 cat）。

通常遇见这种错误
    Indentation Error: expected an indented block
表示你没缩进，编译器没检测到循环语句块。
这里也展现出了 C/C++ 与 python 的不同，没事别乱缩进，就算代码排版难看也别缩进。。。
补充：真要缩进，打 4 个空格也行。（遵循 PEP8 协议）

2.11 切片（ python 术语，意为处理列表的部分元素）
其语法形式为：
Array = [‘gan’,’G’,’GA’,’GAN’,’GANR’]
print(Array[1:4]) //从第 1 个开始，到第 4 个结束（不包含第 4 个）。注意，从 0 开始计算。
out：
[‘G’,’GA’,’GAN’]
如果没有指定开头索引值（如 Array[:4]）,则将从表头开始，其输出结果为：
[‘gan’,’G’,’GA’,’GAN’]
类似的，没有指定末尾索引值（如 Array[1:]）,则从指定索引序号到列表末尾，其输出结果为：
[‘G’,’GA’,’GAN’,’GANR’]
此外，索引序号可以为负数，请注意若为 [-3:]（负几都一样）,并不会给你打完一整个列表，而只会打印最后3个元素，直至列表结束。

切片的遍历(#qiepian)
很简单，加个循环语句（如 for ）就行。
ex：
for cat in cats[:3]:
out：
[前面三个元素]

2.13 列表的复制
其实是对切片的特殊使用，形式为 [:]，意为从 0 开始，直至列表末尾（就是复制整个列表）。
ex：
my_cars = friend_cars[:]
out：
得到两个一毛一样的列表（但是互相独立）

注意不要这么写：
    my_cars = friend_cars
这只是为 my_cars 另起一个变量名而已，这两个指代的还是同一个列表。

2.14 元组
2.14.1 元组的定义
列表中的元素可变，而元组不能变，其他相同。
存在的意义：在整个程序生命周期中保持一组元素值不变。（说白了就是常量）
其定义格式为：
dimensions = (200, 50) //用小括号，不再用中括号
print(dimensions[0])
print(dimensions[1])
out：
200
50

2.14.2 元组的遍历
    ex：
        dimensions = (200, 50)
        for dimension in dimensions:
            print(dimension)
    out：
        200
        50

2.14.3 修改元组的变量
    虽然元组元素不可改变，但是可以给存储元组的变量赋值。
    ex：
        dimensions = (200, 50)
        print("原来的：")
        for dimension in dimensions:
            print(dimension)
        dimensions = (400, 100)
        print("\n改变：")
        for dimension in dimensions:
            print(dimension)
    out：
        原来的：
        200
        50

        改变：
        400
        100

    注意：这里说的修改不是这么改：
        dimensions[0] = 250
        这是改不了的，元组存储的元素不可改变，只能从根源上改。
