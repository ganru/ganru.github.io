列表的使用（大部分情况下把它当作数组来用）

什么是列表（和 C/C++ 中的数组有点像）

存储多个数据的元素集合。

列表元素的访问

ex：
    motorcycles = ['honda', 'yamaha', 'suzuki']
    print(motorcycles)

元素的索引从 0 开始

标注：PY有一种特殊的语法，就是索引可以为负数（如：-1），-1 就是索引列表倒数第一个元素。

列表的增、删、改、查

增

有两种常见方法，append(b)（加在列表尾）、insert(a,b)（a是索引序号，b是新增元素）

删

del语句。如：del Array[0]（删掉列表头部元素）

标注：删还有一种特殊方式 pop() （弹出），这是方法,默认弹出栈顶元素，若要弹出指定元素，只需在括号中添加索引序号。

ex：
    motorcycles = ['honda', 'yamaha', 'suzuki']
    print(motorcycles)
    popped_motorcycle = motorcycles.pop()
    print(motorcycles)
    print(popped_motorcycle)
out：
    ['honda', 'yamaha', 'suzuki']
    ['honda', 'yamaha']
    suzuki

这两种删，前者为销毁，后者为数据提取。

补充：
如果不知道要删掉的元素位置，只知道元素值，用方法 remove(b)（b是要删除的元素值）。
ex：Array.remove(‘doubi’)

改

ex：
    Array = ['gan','G','GG']
    Array[0] = 'gandalf'
out：
    ['gandalf','G','GG']

查

与 切片的遍历 类似。

列表排序与长度确定（注意分永久和暂时排序）

永久性排序：

ex：
    cars = ['bmw', 'audi', 'toyota', 'subaru']
    cars.sort()        //若要大到小排序，在括号内输入 **reverse = True。**
    print(cars)        //如果要反转列表，写 **cars.reverse**
out：
    ['audi', 'bmw', 'subaru', 'toyota']

暂时性排序：

ex：
    Array = ['qwd','asd','gwa','xdf']
    print(Array)
    print(sorted(Array))    //注：如果想大到小排序，写 **sorted(Array，reverse = True)。**
    print(Array)
out：
    ['qwd','asd','gwa','xdf']
    ['asd','gwa','qwd','xdf']
    ['qwd','asd','gwa','xdf']

补充：
首字母大到小排序：reverse = True
字符串长度大到小排序：key = len
key 有很多种方式，具体不表。

列表长度确定

ex：
    cars = ['bmw', 'audi', 'toyota', 'subaru']
    len(cars)

列表使用时可能会产生索引错误

如：索引超过当前列表长度的序值。
列表为空时索引 -1。

列表中 for 的使用

跟 C、C++ 一样，用 for 语句循环遍历，不过 python 的使用方式有所不同。

如 for cat in cats:
        .
        .        //循环语句块（带上缩进才算，即写完 for 换行之后，缩进该行）
        .

表示从列表 cats 选取第一个元素并赋予 cat，然后执行语句块，直至选取到列表 cats 中最后一个元素。

注意这里的命名规则，尽量用复数名词表示列表（如 cats），用单数名词表示单个列表元素（如 cat）。

通常遇见这种错误
Indentation Error: expected an indented block
表示你没缩进，编译器没检测到循环语句块。
这里也展现出了 C/C++ 与 python 的不同，没事别乱缩进，就算代码排版难看也别缩进。。。
补充：真要缩进，打 4 个空格也行。（遵循 PEP8 协议）

函数 range(a,b,c)

作用：生成一系列数字。
使用方式：
从 a 数值开始，至 b 结束，步长值为 c 。（ c 不写则默认为 1 ）

函数 list()

作用：生成列表。

ex：
    numbers = list(range(1,4))
    print(numbers)
out：
    [1,2,3]

数字列表的统计计算

min()、max()、sum() 的使用

列表解析

用于简化代码。

ex：
    numbers = [value**2 for value in range(1,4)]
    print(numbers)
out：
    [1,4,9]

切片（ python 术语，意为处理列表的部分元素）

其语法形式为：

    Array = ['gan','G','GA','GAN','GANR']
    print(Array[1:4])    //从第 1 个开始，到第 4 个结束（不包含第 4 个）。注意，从 0 开始计算。
out：
    ['G','GA','GAN']

如果没有指定开头索引值（如 Array[:4]）,则将从表头开始，其输出结果为：

['gan','G','GA','GAN']

类似的，没有指定末尾索引值（如 Array[1:]）,则从指定索引序号到列表末尾，其输出结果为：

['G','GA','GAN','GANR']

此外，索引序号可以为负数，请注意若为 [-3:]（负几都一样）,并不会给你打完一整个列表，而只会打印最后3个元素，直至列表结束。

切片的遍历

很简单，加个循环语句（如 for ）就行。

    ex：
        for cat in cats[:3]:
    out：
        [前面三个元素]

列表的复制

其实是对切片的特殊使用，形式为 [:]，意为从 0 开始，直至列表末尾（就是复制整个列表）。

ex：
    my_cars = friend_cars[:]
out：
    得到两个一毛一样的列表（但是互相独立）

注意不要这么写：

`my_cars = friend_cars`

这只是为 my_cars 另起一个变量名而已，这两个指代的还是同一个列表。

元组

元组的定义

列表中的元素可变，而元组不能变，其他相同。

存在的意义：在整个程序生命周期中保持一组元素值不变。（说白了就是常量）
其定义格式为：

    dimensions = (200, 50)        //用小括号，不再用中括号
    print(dimensions[0])
    print(dimensions[1])
out：
    200
    50

元组的遍历

ex：
    dimensions = (200, 50)
    for dimension in dimensions:
        print(dimension)
out：
    200
    50

修改元组的变量

虽然元组元素不可改变，但是可以给存储元组的变量赋值。

ex：
    dimensions = (200, 50)
    print("原来的：")
    for dimension in dimensions:
        print(dimension)
    dimensions = (400, 100)
    print("\n改变：")
    for dimension in dimensions:
        print(dimension)
out：
    原来的：
    200
    50

    改变：
    400
    100

注意：这里说的修改不是这么改：
dimensions[0] = 250
这是改不了的，元组存储的元素不可改变，只能从根源上改。

  2. for 的使用（和 C 有所不同，而且密切注意缩进的不同）
  3. 列表的复制
  4. 元组的应用

字典

字典的定义

类似于数据结构中的线性表，链表之类，具有将数据关联起来的作用。

字典的格式：

student_0 = {'name':'ganru','age':19} //由多个键值对构成

使用字典来存储用户提供的数据或在编写能自动生成大量键—值对的代码时，通常都需要先定义一个空字典。

字典的访问

ex：
    student_0 = {'name':'ganru','age':19}
    print(student_0['name'])
out：
    ganru

字典的增、删、改、查

增

ex：
    student_0 = {'name':'ganru','age':19}
    print(student_0)
    student_0['sex'] = 'man'
    student_0['graduated school'] = 'JYU'
    print(student_0)
out：
    {'name':'ganru','age':19}
    {'name':'ganru','age':19,'sex':'man','graduated school':'JYU'}

删

用 del 语句。

ex：
    student_0 = {'name':'ganru','age':19,'sex':'man','graduated school':'JYU'}
    print(student_0)
    del student_0['graduated school']
    print(student_0)
out：
    {'name':'ganru','age':19,'sex':'man','graduated school':'JYU'}
    {'name':'ganru','age':19,'sex':'man'}

注意：永删。

改

ex：
    student_0 = {'name':'ganru'}
    print("my name is" + student_0['name'] + ".")
    student_0['name'] = 'Gandalf'
    print("my name is" + student_0['name'] + ".")
out：
    my name is gan.
    my name is gandalf.

查

与 字典的访问 相似。

字典的遍历

遍历所有键值对

用 for 循环。

ex：
    student_0 = {
        'name':'ganru',
        'age':19,
        }
    for key,value in student_0.items()：    //声明两个变量，用于存储键值对中的键和值。
        print("\nKey: " + key)
        print("Value: " + value)
out：
    Key: name
    Value: ganru

    Key: age
    Value: 19

注意：遍历字典时，键值对的返回顺序与存储顺序不一定相同。

遍历无值键

用方法 keys() 来对字典进行遍历。

ex：
    student_0 = {
        'name':'ganru',
        'age':19,
        'sex':'man',
        'graduated school':'JYU',
        }
    for name in student_0.keys():        //其实可以不写 .keys()，输出依然不变（默认遍历）
        print(name.title())
out：
    name
    age
    graduated school

以某种顺序遍历字典中的所有键

用函数 sorted() 。

ex：
    favorite_languages = {
        'jen': 'python',
        'sarah': 'c',
        'edward': 'ruby',
        'phil': 'python',
        }

    for name in sorted(favorite_languages.keys()):
        print(name.title() + ", thank you for taking the poll.")
out：
    Edward, thank you for taking the poll.
    Jen, thank you for taking the poll.
    Phil, thank you for taking the poll.
    Sarah, thank you for taking the poll.

遍历所有值（无键）

用方法 values()。（可能得到很多重复项）

ex：
    favorite_languages = {
        'jen': 'python',
        'sarah': 'c',
        'edward': 'ruby',
        'phil': 'python',
        }

    for language in favorite_languages.values():
        print(language.title())
out：
    Python
    C
    Ruby
    Python

在 values() 的基础上用集合set()，可除掉重复项。

ex：
    favorite_languages = {
        'jen': 'python',
        'sarah': 'c',
        'edward': 'ruby',
        'phil': 'python',
        }

    for language in set(favorite_languages.values()):
        print(language.title())
out：
    Python
    C
    Ruby

字典的嵌套

比如 字典中的字典。

ex：
    alien_0 = {'color': 'green', 'points': 5}
    alien_1 = {'color': 'yellow', 'points': 10}
    alien_2 = {'color': 'red', 'points': 15}
    aliens = [alien_0, alien_1, alien_2]
    for alien in aliens:
        print(alien)
out：
    {'color': 'green', 'points': 5}
    {'color': 'yellow', 'points': 10}
    {'color': 'red', 'points': 15}

字典中存储列表

ex：
    student_0 = {
        'name': 'ganru',
        'language': ['python', 'C', 'C++'],
        }


    print(student_0['name'].title() + "'s favorite languages are:")
    for language in student_0['language']:
        print("\t" + language.title())

out：
    Ganru's favorite languages are:
        Python
        C
        C++

学习要点

  1. 如何定义字典，以及如何使用存储在字典中的信息；
  2. 如何访问和修改字典中的元素，以及如何遍历字典中的所有信息；
  3. 如何遍历字典中所有的键值对、所有的键和所有的值；
  4. 如何在列表中嵌套字典、在字典中嵌套列表以及在字典中嵌套字典。

用户输入与 while 循环

用户输入

函数 input() 的使用（类似于 gets）

作用：接收用户键入的数据并保存为字符串。

ex：
    message = input("Tell me something, and I will repeat it back to you: ")
    print(message)

out：
    Tell me something, and I will repeat it back to you: Hello everyone!
    Hello everyone!

input() 括号里面填写的是提示信息，给用户指示该如何键入。

有时候如果提示信息太长，一行写不完，可以这么做：

ex：
    prompt = "If you tell us who you are, we can personalize the messages you see."
    prompt += "\nWhat is your first name? "

    name = input(prompt)
    print("\nHello, " + name + "!")

out：
    If you tell us who you are, we can personalize the messages you see.
    What is your first name? Ganru
    Hello, Ganru!

可看出：采用的是字符串拼接的方式。

注意：这里 Python 接收到的用户键入的数据永远是字符串。

函数 int() 的使用（类似于 putchar）

常见作用：

  1. 接收用户键入的数据并保存为整型数值。（通常配合 input() 使用）

  2. 将字符串转变为数值表示。

    ex：

    height = input("How tall are you, in inches? ")
height = int(height)

if height >= 36:
    print("\nYou're tall enough to ride!")
else:
    print("\nYou'll be able to ride when you're a little older.")

    out：

    How tall are you, in inches? 71
You're tall enough to ride!

ex：

运行时中断 （break 语句的使用）

常见作用：

  1. 让机器判断并跳出无限循环。

  2. 让机器筛选特定值。

    ex：

    prompt = "\nwho are you?"
message = ""
while True:
    message = input(prompt)

    if message == 'ganru':
        break
    else:
        print("Hello, " + message.title() + "!")

ex：

    while current_number < 10:
     current_number += 1

        if current_number % 2 == 0:
            continue

        print(current_number)        //打印奇数

while 与列表、字典的关系

for 适合遍历列表，while 适合处理列表。

.
.

列表间元素移动

ex：
    unconfirmed_users = ['alice', 'brian', 'candace']
    confirmed_users = []

    while unconfirmed_users:
        current_user = unconfirmed_users.pop()

        print("Verifying user: " + current_user.title())
        confirmed_users.append(current_user)

    print("\nThe following users have been confirmed:")
    for confirmed_user in confirmed_users:
        print(confirmed_user.title())
out：
    Verifying user: Candace
    Verifying user: Brian
    Verifying user: Alice

    The following users have been confirmed:
    Candace
    Brian
    Alice

while 与 remove

ex：
    names = ['ganru','gandalf','gan','ganru','gandalf','ganru','gandalf']
    print(names)

    while 'gandalf' in names:    //判断字符串 'gandalf' 是否在 names 列表中。
        names.remove('gandalf')

    print(names)
out：
    ['ganru','gandalf','gan','ganru','gandalf','ganru','gandalf']
    ['ganru','gan','ganru','ganru',]

使用用户输入来填充字典

ex：
    responses = {}

    # 设置一个标志，指出调查是否继续
    polling_active = True

    while polling_active:
        # 提示输入被调查者的名字和回答
        name = input("\nWhat is your name? ")
        response = input("Which mountain would you like to climb someday? ")

        # 将答卷存储在字典中
        responses[name] = response

        # 看看是否还有人要参与调查
        repeat = input("Would you like to let another person respond? (yes/ no) ")
        if repeat == 'no':
            polling_active = False

        # 调查结束，显示结果
        print("\n--- Poll Results ---")
        for name, response in responses.items():
            print(name + " would like to climb " + response + ".")
