函数的定义

关键字 def 定义函数。

ex：
    def hello(username):
        """此处写函数注释，建议每次都写，方便日后维护"""
        print(username.title() + ", Welcome to you!")

    hello('ganru')    #要加单引号，不然编译器会检测为未定义对象而不是字符串变量。

out：
    ganru, Welcome to you!

实参与形参

如 hello() 的函数定义，括号里面的变量 username 是形参，只具有形式意义，而值 ‘ganru’ 是实参，具有实际意义。
如何判断：函数定义中的变量为形参，函数调用中的变量为实参。

实参的传递

位置实参

定义：基于实参的顺序将函数调用中的每个实参都关联到函数定义中的一个形参。

ex：
    def student(name, sex):
        .
        .
        .

    student('garu','boy')    #这里的 'garu' 为第一顺序，'boy' 为第二顺序。

关键字实参

定义：传递给函数的名称—值对。无需考虑函数调用中的实参顺序，还清楚地指出了函数调用中各个值的用途。

ex：
    def student(name, sex):
        .
        .
        .

    student(sex = 'boy', name = 'garu')    #顺序反了也没事

注意：使用关键字实参时，务必准确地指定函数定义中的形参名。

默认值

调用函数中给形参提供了实参时，Python 将使用指定的实参值；否则，将使用形参的默认值。
作用：给形参指定默认值后，可在函数调用中省略相应的实参。

ex：
    def student(name, sex = 'boy'):
        .
        .
        .

    student(name = 'garu')    #以后每次调用，如果没有显式提供实参给 sex，sex 都是 'boy'。

注意：如果以后要用默认值，也就是考虑到可能不写值，则形参放最后面（如：sex），以免出现位置实参错误。

6.5 实参可选

ex：

这种函数或许适用大多数情况，但是直辖市啥整？

def location(country, city, province = ''):    # province 放后面，以免出现位置实参错误
    """个人居住地"""
    if province:    #如果是空字符串，Python 将解读为 False。
        my_location = country + ' ' + province + ' ' + city
    else:
        my_location = country + ' ' + city
    return my_location.title()

新函数就是所说的实参可选。

任意数量的实参传递

传递任意数量的实参 （建立空元组）

常见作用：
有时候真不知道函数要接收多少个实参，这时候可以对形参名加个 * ，让 Python 创建一个空元组，并将收到的所有值都封装到这个元组中。

ex：
    def make_pizza(*toppings):
        """概述要制作的比萨"""
        print("\nMaking a pizza with the following toppings:")
        for topping in toppings:
            print("- " + topping)

    make_pizza('pepperoni')
    make_pizza('mushrooms', 'green peppers', 'extra cheese')

函数调用解析：
这时候 * 号调用作用于 tuple/list/str 前，意思是解包(unpack)tuple/list/str，解包后的 tuple/list/str 将不再是一个参数，而是每个元素都是一个函数参数，依次传递给函数。

out：
    Making a pizza with the following toppings:
    - pepperoni
    Making a pizza with the following toppings:
    - mushrooms
    - green peppers
    - extra cheese

结合使用位置实参和任意数量实参

ex：
    def make_pizza(size, *toppings):
        """概述要制作的比萨"""
        print("\nMaking a " + str(size) +
            "-inch pizza with the following toppings:")
        for topping in toppings:
            print("- " + topping)

    make_pizza(16, 'pepperoni')
    make_pizza(12, 'mushrooms', 'green peppers', 'extra cheese')
out：
    Making a 16-inch pizza with the following toppings:
    - pepperoni

    Making a 12-inch pizza with the following toppings:
    - mushrooms
    - green peppers
    - extra cheese

使用任意数量的关键字实参 （建立空字典）

预先不知道传递给函数的会是什么样的信息，有多少个，这时可以用关键字实参调用。即接收任意数量的 键-值 对。
格式：对形参加 ** ，让Python创建一个空字典，并将收到的所有 键-值 对都封装到这个字典中。

ex：
    def build_profile(first, last, **user_info):    #**号创建空字典
        """创建一个字典，其中包含我们知道的有关用户的一切"""
        profile = {}
        profile['first_name'] = first
        profile['last_name'] = last
        for key, value in user_info.items():
            profile[key] = value
        return profile

    user_profile = build_profile('albert', 'einstein',
                                location='princeton',
                                field='physics')
    print(user_profile)
out：
    {'first_name': 'albert', 'last_name': 'einstein', 'location': 'princeton', 'field': 'physics'}

字典的定义

类似于数据结构中的线性表，链表之类，具有将数据关联起来的作用。

字典的格式：

student_0 = {'name':'ganru','age':19} //由多个键值对构成

使用字典来存储用户提供的数据或在编写能自动生成大量键—值对的代码时，通常都需要先定义一个空字典。

字典的访问

ex：
    student_0 = {'name':'ganru','age':19}
    print(student_0['name'])
out：
    ganru

字典的增、删、改、查

增

ex：
    student_0 = {'name':'ganru','age':19}
    print(student_0)
    student_0['sex'] = 'man'
    student_0['graduated school'] = 'JYU'
    print(student_0)
out：
    {'name':'ganru','age':19}
    {'name':'ganru','age':19,'sex':'man','graduated school':'JYU'}

删

用 del 语句。

ex：
    student_0 = {'name':'ganru','age':19,'sex':'man','graduated school':'JYU'}
    print(student_0)
    del student_0['graduated school']
    print(student_0)
out：
    {'name':'ganru','age':19,'sex':'man','graduated school':'JYU'}
    {'name':'ganru','age':19,'sex':'man'}

注意：永删。

改

ex：
    student_0 = {'name':'ganru'}
    print("my name is" + student_0['name'] + ".")
    student_0['name'] = 'Gandalf'
    print("my name is" + student_0['name'] + ".")
out：
    my name is gan.
    my name is gandalf.

查

与 字典的访问 相似。

字典的遍历

遍历所有键值对

用 for 循环。

ex：
    student_0 = {
        'name':'ganru',
        'age':19,
        }
    for key,value in student_0.items()：    //声明两个变量，用于存储键值对中的键和值。
        print("\nKey: " + key)
        print("Value: " + value)
out：
    Key: name
    Value: ganru

    Key: age
    Value: 19

注意：遍历字典时，键值对的返回顺序与存储顺序不一定相同。

遍历无值键

用方法 keys() 来对字典进行遍历。

ex：
    student_0 = {
        'name':'ganru',
        'age':19,
        'sex':'man',
        'graduated school':'JYU',
        }
    for name in student_0.keys():        //其实可以不写 .keys()，输出依然不变（默认遍历）
        print(name.title())
out：
    name
    age
    graduated school

以某种顺序遍历字典中的所有键

用函数 sorted() 。

ex：
    favorite_languages = {
        'jen': 'python',
        'sarah': 'c',
        'edward': 'ruby',
        'phil': 'python',
        }

    for name in sorted(favorite_languages.keys()):
        print(name.title() + ", thank you for taking the poll.")
out：
    Edward, thank you for taking the poll.
    Jen, thank you for taking the poll.
    Phil, thank you for taking the poll.
    Sarah, thank you for taking the poll.

遍历所有值（无键）

用方法 values()。（可能得到很多重复项）

ex：
    favorite_languages = {
        'jen': 'python',
        'sarah': 'c',
        'edward': 'ruby',
        'phil': 'python',
        }

    for language in favorite_languages.values():
        print(language.title())
out：
    Python
    C
    Ruby
    Python

在 values() 的基础上用集合set()，可除掉重复项。

ex：
    favorite_languages = {
        'jen': 'python',
        'sarah': 'c',
        'edward': 'ruby',
        'phil': 'python',
        }

    for language in set(favorite_languages.values()):
        print(language.title())
out：
    Python
    C
    Ruby

if 语句的使用

cars = ['audi', 'bmw', 'subaru', 'toyota']
for car in cars:
    if car == 'bmw':
        print(car.upper())
    elif car == 'audi':
        print(car.title() + " is very good!")                
    else:
        print(car.title())
out：
    Audi is very good!
    BMW
    Subaru
    Toyota

常用的判断符号

== 判断是否相等
!= 判断是否不等
and 判断两条件是否同时为 True 。（相当于 && ）

什么是列表（和 C/C++ 中的数组有点像）

存储多个数据的元素集合。

列表元素的访问

ex：
    motorcycles = ['honda', 'yamaha', 'suzuki']
    print(motorcycles)

元素的索引从 0 开始

标注：PY有一种特殊的语法，就是索引可以为负数（如：-1），-1 就是索引列表倒数第一个元素。

列表的增、删、改、查

增

有两种常见方法，append(b)（加在列表尾）、insert(a,b)（a是索引序号，b是新增元素）

删

del语句。如：del Array[0]（删掉列表头部元素）

标注：删还有一种特殊方式 pop() （弹出），这是方法,默认弹出栈顶元素，若要弹出指定元素，只需在括号中添加索引序号。

ex：
    motorcycles = ['honda', 'yamaha', 'suzuki']
    print(motorcycles)
    popped_motorcycle = motorcycles.pop()
    print(motorcycles)
    print(popped_motorcycle)
out：
    ['honda', 'yamaha', 'suzuki']
    ['honda', 'yamaha']
    suzuki

这两种删，前者为销毁，后者为数据提取。

补充：
如果不知道要删掉的元素位置，只知道元素值，用方法 remove(b)（b是要删除的元素值）。
ex：Array.remove(‘doubi’)

改

ex：
    Array = ['gan','G','GG']
    Array[0] = 'gandalf'
out：
    ['gandalf','G','GG']

查

与 切片的遍历 类似。

列表排序与长度确定（注意分永久和暂时排序）

永久性排序：

ex：
    cars = ['bmw', 'audi', 'toyota', 'subaru']
    cars.sort()        //若要大到小排序，在括号内输入 **reverse = True。**
    print(cars)        //如果要反转列表，写 **cars.reverse**
out：
    ['audi', 'bmw', 'subaru', 'toyota']

暂时性排序：

ex：
    Array = ['qwd','asd','gwa','xdf']
    print(Array)
    print(sorted(Array))    //注：如果想大到小排序，写 **sorted(Array，reverse = True)。**
    print(Array)
out：
    ['qwd','asd','gwa','xdf']
    ['asd','gwa','qwd','xdf']
    ['qwd','asd','gwa','xdf']

补充：
首字母大到小排序：reverse = True
字符串长度大到小排序：key = len
key 有很多种方式，具体不表。

列表长度确定

ex：
    cars = ['bmw', 'audi', 'toyota', 'subaru']
    len(cars)

列表使用时可能会产生索引错误

如：索引超过当前列表长度的序值。
列表为空时索引 -1。

列表中 for 的使用

跟 C、C++ 一样，用 for 语句循环遍历，不过 python 的使用方式有所不同。

如 for cat in cats:
        .
        .        //循环语句块（带上缩进才算，即写完 for 换行之后，缩进该行）
        .

表示从列表 cats 选取第一个元素并赋予 cat，然后执行语句块，直至选取到列表 cats 中最后一个元素。

注意这里的命名规则，尽量用复数名词表示列表（如 cats），用单数名词表示单个列表元素（如 cat）。

通常遇见这种错误
Indentation Error: expected an indented block
表示你没缩进，编译器没检测到循环语句块。
这里也展现出了 C/C++ 与 python 的不同，没事别乱缩进，就算代码排版难看也别缩进。。。
补充：真要缩进，打 4 个空格也行。（遵循 PEP8 协议）

切片（ python 术语，意为处理列表的部分元素）

其语法形式为：

    Array = ['gan','G','GA','GAN','GANR']
    print(Array[1:4])    //从第 1 个开始，到第 4 个结束（不包含第 4 个）。注意，从 0 开始计算。
out：
    ['G','GA','GAN']

如果没有指定开头索引值（如 Array[:4]）,则将从表头开始，其输出结果为：

['gan','G','GA','GAN']

类似的，没有指定末尾索引值（如 Array[1:]）,则从指定索引序号到列表末尾，其输出结果为：

['G','GA','GAN','GANR']

此外，索引序号可以为负数，请注意若为 [-3:]（负几都一样）,并不会给你打完一整个列表，而只会打印最后3个元素，直至列表结束。

切片的遍历

很简单，加个循环语句（如 for ）就行。

    ex：
        for cat in cats[:3]:
    out：
        [前面三个元素]

列表的复制

其实是对切片的特殊使用，形式为 [:]，意为从 0 开始，直至列表末尾（就是复制整个列表）。

ex：
    my_cars = friend_cars[:]
out：
    得到两个一毛一样的列表（但是互相独立）

注意不要这么写：

`my_cars = friend_cars`

这只是为 my_cars 另起一个变量名而已，这两个指代的还是同一个列表。

元组

元组的定义

列表中的元素可变，而元组不能变，其他相同。

存在的意义：在整个程序生命周期中保持一组元素值不变。（说白了就是常量）
其定义格式为：

    dimensions = (200, 50)        //用小括号，不再用中括号
    print(dimensions[0])
    print(dimensions[1])
out：
    200
    50

元组的遍历

ex：
    dimensions = (200, 50)
    for dimension in dimensions:
        print(dimension)
out：
    200
    50

修改元组的变量

虽然元组元素不可改变，但是可以给存储元组的变量赋值。

ex：
    dimensions = (200, 50)
    print("原来的：")
    for dimension in dimensions:
        print(dimension)
    dimensions = (400, 100)
    print("\n改变：")
    for dimension in dimensions:
        print(dimension)
out：
    原来的：
    200
    50

    改变：
    400
    100

注意：这里说的修改不是这么改：
dimensions[0] = 250
这是改不了的，元组存储的元素不可改变，只能从根源上改。
