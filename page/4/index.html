类的定义

可以理解为 C 中的结构体，两者有点像，但又有不同。
根据类来创建对象被称为实例化，这让你能够使用类的实例。

类的创建与使用

创建类

创建一个学生类，赋予每个学生其学习（study()）和睡觉（sleep()）的能力。

ex：

class Student():    #首字母要大写，指的是类（约定的），括号空，是因为要从空白建立类
    """学生类的模拟"""

    def \_\_init\_\_(self, name, age):    
    #注意这里有个坑，默认方法构造书写格式是__init__，而不是_init_，即在init两侧都是双下划线，不是单下划线。 
        """初始化学生属性"""
        self.name = name
        self.age = age

    def study(self):
        """模拟学生学习"""
        print(self.name.title() + " is now learning.")

    def sleep(self):
        """模拟学生睡觉"""
        print(self.name.title() + " is now sleeping.")

    def get_descriptive(self):
        """返回学生信息"""
        descriptive = self.name + ' ' + str(self.age)
        return descriptive.title()
  1. 上述代码中，在类中出现的函数称为方法，如 3 个 def，与函数一样，只是名称发生了变化。
  2. 在类中，每当使用者用 Student 类创建新实例时，Python 就会自动运行方法 __init__()。每次写类都要初始化。
  3. 方法也有不同，方法名首尾都有下划线的一般指 默认方法，这是为了避免Python默认方法与普通方法发生名称冲突。
  4. 在 init() 中 self 必不可少，Python 调用这个方法来创建 Student 实例时，将自动传入实参 self。
  5. 每个与类相关联的方法调用都自动传递实参 self，它是一个指向实例本身的引用，让实例能够访问类中的属性和方法。
  6. 可通过实例访问的变量成为 属性（如：name、age）。
  7. 每次新建实例时，只需写后面两个形参值。
  8. 另外两个方法，由于不需要额外的信息，故只有一个形参 self。
  9. 虽然没有写显式 return 语句，但 Python 会自动返回一个表示该学生的实例。

类的继承

有时候懒得写多几个重复项，想引用其他类并自定义自己的类，可用继承。原有类称父类，新类称子类。

父类：

class Car():
    """一次模拟汽车的简单尝试"""
    def __init__(self, make, model, year):
        self.make = make
        self.model = model
        self.year = year
        self.odometer_reading = 0

    def get_descriptive_name(self):
        long_name = str(self.year) + ' ' + self.make + ' ' + self.model
        return long_name.title()

    def read_odometer(self):
        print("This car has " + str(self.odometer_reading) + " miles on it.")

    def update_odometer(self, mileage):
        if mileage >= self.odometer_reading:
            self.odometer_reading = mileage
        else:
            print("You can't roll back an odometer!")

    def increment_odometer(self, miles):
        self.odometer_reading += miles

子类：

class ElectricCar(Car):        #类括号内若导入参数，则继承某一个类
    """电动汽车的独特之处"""

    def __init__(self, make, model, year):
        """初始化父类的属性"""
        super().__init__(make, model, year)        
        #super 是一个特殊函数，让 Python 调用 ElectricCar 的父类的方法 __init__()，让 ElectricCar 实例包含父类的所有属性。


my_tesla = ElectricCar('tesla', 'model s', 2016)
print(my_tesla.get_descriptive_name())

out：

2016 Tesla Model S
  1. 父类和子类必须要在同一文件中，且父类在子类前面。
  2. 父类也称超类，名称 super 因此而得名。

类的导入

跟函数模块差不多，将类写入一个 .py 文件。
导入格式：

import module_name（.py 文件名）
from module_name（.py 文件名） import class_name（在 .py 文件中的类名）
from module_name import class_0, class_1, …

如果模块与模块之间有联系，则这两个模块都要导入同一个文件中，不然创建实例时将引发错误。

Python 标准库

Python标准库是一组模块，安装的Python都包含它。大致理解类之后，就可以使用他人的模块了。

类编码风格

  1. 驼峰命名法：类名中的每个单词的首字母都大写，而不使用下划线。
    ex：
    class Car():
    class Student():
    …

  2. 实例名和模块名都采用小写格式，并在单词之间加上下划线。
    ex：
    my_tesla
    student_0
    son.py
    …

  3. 对于每个类，都写类注释。

  4. 在类中，可使用一个空行来分隔方法；而在模块中，可使用两个空行来分隔类。
  5. 需要同时导入标准库中的模块和你编写的模块时，先编写导入标准库模块的 import 语句，再添加一个空行，然后编写导入你自己编写的模块的 import 语句。

函数的定义

关键字 def 定义函数。

ex：
    def hello(username):
        """此处写函数注释，建议每次都写，方便日后维护"""
        print(username.title() + ", Welcome to you!")

    hello('ganru')    #要加单引号，不然编译器会检测为未定义对象而不是字符串变量。

out：
    ganru, Welcome to you!

实参与形参

如 hello() 的函数定义，括号里面的变量 username 是形参，只具有形式意义，而值 ‘ganru’ 是实参，具有实际意义。
如何判断：函数定义中的变量为形参，函数调用中的变量为实参。

实参的传递

位置实参

定义：基于实参的顺序将函数调用中的每个实参都关联到函数定义中的一个形参。

ex：
    def student(name, sex):
        .
        .
        .

    student('garu','boy')    #这里的 'garu' 为第一顺序，'boy' 为第二顺序。

关键字实参

定义：传递给函数的名称—值对。无需考虑函数调用中的实参顺序，还清楚地指出了函数调用中各个值的用途。

ex：
    def student(name, sex):
        .
        .
        .

    student(sex = 'boy', name = 'garu')    #顺序反了也没事

注意：使用关键字实参时，务必准确地指定函数定义中的形参名。

默认值

调用函数中给形参提供了实参时，Python 将使用指定的实参值；否则，将使用形参的默认值。
作用：给形参指定默认值后，可在函数调用中省略相应的实参。

ex：
    def student(name, sex = 'boy'):
        .
        .
        .

    student(name = 'garu')    #以后每次调用，如果没有显式提供实参给 sex，sex 都是 'boy'。

注意：如果以后要用默认值，也就是考虑到可能不写值，则形参放最后面（如：sex），以免出现位置实参错误。

任意数量的实参传递

传递任意数量的实参 （建立空元组）

常见作用：
有时候真不知道函数要接收多少个实参，这时候可以对形参名加个 * ，让 Python 创建一个空元组，并将收到的所有值都封装到这个元组中。

ex：
    def make_pizza(*toppings):
        """概述要制作的比萨"""
        print("\nMaking a pizza with the following toppings:")
        for topping in toppings:
            print("- " + topping)

    make_pizza('pepperoni')
    make_pizza('mushrooms', 'green peppers', 'extra cheese')

函数调用解析：
这时候 * 号调用作用于 tuple/list/str 前，意思是解包(unpack)tuple/list/str，解包后的 tuple/list/str 将不再是一个参数，而是每个元素都是一个函数参数，依次传递给函数。

out：
    Making a pizza with the following toppings:
    - pepperoni
    Making a pizza with the following toppings:
    - mushrooms
    - green peppers
    - extra cheese

结合使用位置实参和任意数量实参

ex：
    def make_pizza(size, *toppings):
        """概述要制作的比萨"""
        print("\nMaking a " + str(size) +
            "-inch pizza with the following toppings:")
        for topping in toppings:
            print("- " + topping)

    make_pizza(16, 'pepperoni')
    make_pizza(12, 'mushrooms', 'green peppers', 'extra cheese')
out：
    Making a 16-inch pizza with the following toppings:
    - pepperoni

    Making a 12-inch pizza with the following toppings:
    - mushrooms
    - green peppers
    - extra cheese

使用任意数量的关键字实参 （建立空字典）

预先不知道传递给函数的会是什么样的信息，有多少个，这时可以用关键字实参调用。即接收任意数量的 键-值 对。
格式：对形参加 ** ，让Python创建一个空字典，并将收到的所有 键-值 对都封装到这个字典中。

ex：
    def build_profile(first, last, **user_info):    #**号创建空字典
        """创建一个字典，其中包含我们知道的有关用户的一切"""
        profile = {}
        profile['first_name'] = first
        profile['last_name'] = last
        for key, value in user_info.items():
            profile[key] = value
        return profile

    user_profile = build_profile('albert', 'einstein',
                                location='princeton',
                                field='physics')
    print(user_profile)
out：
    {'first_name': 'albert', 'last_name': 'einstein', 'location': 'princeton', 'field': 'physics'}

导入特定函数及给函数指定别名

格式：

用户输入

函数 input() 的使用（类似于 gets）

有时候如果提示信息太长，一行写不完，可以这么做：

    prompt = "If you tell us who you are, we can personalize the messages you see."

    What is your first name? Ganru
    Hello, Ganru!

使用用户输入来填充字典

    responses = {}

    # 设置一个标志，指出调查是否继续
    polling_active = True

    while polling_active:
        # 提示输入被调查者的名字和回答
        name = input("\nWhat is your name? ")
        response = input("Which mountain would you like to climb someday? ")

        # 将答卷存储在字典中
        responses[name] = response

        # 看看是否还有人要参与调查
        repeat = input("Would you like to let another person respond? (yes/ no) ")
        if repeat == 'no':
            polling_active = False

        # 调查结束，显示结果
        print("\n--- Poll Results ---")
        for name, response in responses.items():
            print(name + " would like to climb " + response + ".")

字典的定义

类似于数据结构中的线性表，链表之类，具有将数据关联起来的作用。

字典的格式：

student_0 = {'name':'ganru','age':19} //由多个键值对构成

使用字典来存储用户提供的数据或在编写能自动生成大量键—值对的代码时，通常都需要先定义一个空字典。

字典的访问

ex：
    student_0 = {'name':'ganru','age':19}
    print(student_0['name'])
out：
    ganru

字典的增、删、改、查

增

ex：
    student_0 = {'name':'ganru','age':19}
    print(student_0)
    student_0['sex'] = 'man'
    student_0['graduated school'] = 'JYU'
    print(student_0)
out：
    {'name':'ganru','age':19}
    {'name':'ganru','age':19,'sex':'man','graduated school':'JYU'}

删

用 del 语句。

ex：
    student_0 = {'name':'ganru','age':19,'sex':'man','graduated school':'JYU'}
    print(student_0)
    del student_0['graduated school']
    print(student_0)
out：
    {'name':'ganru','age':19,'sex':'man','graduated school':'JYU'}
    {'name':'ganru','age':19,'sex':'man'}

注意：永删。

改

ex：
    student_0 = {'name':'ganru'}
    print("my name is" + student_0['name'] + ".")
    student_0['name'] = 'Gandalf'
    print("my name is" + student_0['name'] + ".")
out：
    my name is gan.
    my name is gandalf.

查

与 字典的访问 相似。

字典的遍历

遍历所有键值对

用 for 循环。

ex：
    student_0 = {
        'name':'ganru',
        'age':19,
        }
    for key,value in student_0.items()：    //声明两个变量，用于存储键值对中的键和值。
        print("\nKey: " + key)
        print("Value: " + value)
out：
    Key: name
    Value: ganru

    Key: age
    Value: 19

注意：遍历字典时，键值对的返回顺序与存储顺序不一定相同。

遍历无值键

用方法 keys() 来对字典进行遍历。

ex：
    student_0 = {
        'name':'ganru',
        'age':19,
        'sex':'man',
        'graduated school':'JYU',
        }
    for name in student_0.keys():        //其实可以不写 .keys()，输出依然不变（默认遍历）
        print(name.title())
out：
    name
    age
    graduated school

以某种顺序遍历字典中的所有键

用函数 sorted() 。

ex：
    favorite_languages = {
        'jen': 'python',
        'sarah': 'c',
        'edward': 'ruby',
        'phil': 'python',
        }

    for name in sorted(favorite_languages.keys()):
        print(name.title() + ", thank you for taking the poll.")
out：
    Edward, thank you for taking the poll.
    Jen, thank you for taking the poll.
    Phil, thank you for taking the poll.
    Sarah, thank you for taking the poll.

遍历所有值（无键）

用方法 values()。（可能得到很多重复项）

ex：
    favorite_languages = {
        'jen': 'python',
        'sarah': 'c',
        'edward': 'ruby',
        'phil': 'python',
        }

    for language in favorite_languages.values():
        print(language.title())
out：
    Python
    C
    Ruby
    Python

在 values() 的基础上用集合set()，可除掉重复项。

ex：
    favorite_languages = {
        'jen': 'python',
        'sarah': 'c',
        'edward': 'ruby',
        'phil': 'python',
        }

    for language in set(favorite_languages.values()):
        print(language.title())
out：
    Python
    C
    Ruby
