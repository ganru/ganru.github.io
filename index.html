列表的使用（大部分情况下把它当作数组来用）

什么是列表（和 C/C++ 中的数组有点像）

存储多个数据的元素集合。

列表元素的访问

ex：
    motorcycles = ['honda', 'yamaha', 'suzuki']
    print(motorcycles)

元素的索引从 0 开始

标注：PY有一种特殊的语法，就是索引可以为负数（如：-1），-1 就是索引列表倒数第一个元素。

列表的增、删、改、查

增

有两种常见方法，append(b)（加在列表尾）、insert(a,b)（a是索引序号，b是新增元素）

删

del语句。如：del Array[0]（删掉列表头部元素）

标注：删还有一种特殊方式 pop() （弹出），这是方法,默认弹出栈顶元素，若要弹出指定元素，只需在括号中添加索引序号。

ex：
    motorcycles = ['honda', 'yamaha', 'suzuki']
    print(motorcycles)
    popped_motorcycle = motorcycles.pop()
    print(motorcycles)
    print(popped_motorcycle)
out：
    ['honda', 'yamaha', 'suzuki']
    ['honda', 'yamaha']
    suzuki

这两种删，前者为销毁，后者为暂时删除。

补充：
如果不知道要删掉的元素位置，只知道元素值，用方法 remove(b)（b是要删除的元素值）。
ex：Array.remove(‘doubi’)

改

ex：
    Array = ['gan','G','GG']
    Array[0] = 'gandalf'
out：
    ['gandalf','G','GG']

查

与 切片的遍历 类似。

列表排序与长度确定（注意分永久和暂时排序）

永久性排序：

ex：
    cars = ['bmw', 'audi', 'toyota', 'subaru']
    cars.sort()        //若要大到小排序，在括号内输入 **reverse = True。**
    print(cars)        //如果要反转列表，写 **cars.reverse**
out：
    ['audi', 'bmw', 'subaru', 'toyota']

暂时性排序：

ex：
    Array = ['qwd','asd','gwa','xdf']
    print(Array)
    print(sorted(Array))    //注：如果想大到小排序，写 **sorted(Array，reverse = True)。**
    print(Array)
out：
    ['qwd','asd','gwa','xdf']
    ['asd','gwa','qwd','xdf']
    ['qwd','asd','gwa','xdf']

补充：
首字母大到小排序：reverse = True
字符串长度大到小排序：key = len
key 有很多种方式，具体不表。

列表长度确定

ex：
    cars = ['bmw', 'audi', 'toyota', 'subaru']
    len(cars)

列表使用时可能会产生索引错误

如：索引超过当前列表长度的序值。
列表为空时索引 -1。

列表中 for 的使用

跟 C、C++ 一样，用 for 语句循环遍历，不过 python 的使用方式有所不同。

如 for cat in cats:
        .
        .        //循环语句块（带上缩进才算，即写完 for 换行之后，缩进该行）
        .

表示从列表 cats 选取第一个元素并赋予 cat，然后执行语句块，直至选取到列表 cats 中最后一个元素。

注意这里的命名规则，尽量用复数名词表示列表（如 cats），用单数名词表示单个列表元素（如 cat）。

通常遇见这种错误
Indentation Error: expected an indented block
表示你没缩进，编译器没检测到循环语句块。
这里也展现出了 C/C++ 与 python 的不同，没事别乱缩进，就算代码排版难看也别缩进。。。
补充：真要缩进，打 4 个空格也行。（遵循 PEP8 协议）

函数 range(a,b,c)

作用：生成一系列数字。
使用方式：
从 a 数值开始，至 b 结束，步长值为 c 。（ c 不写则默认为 1 ）

函数 list()

作用：生成列表。

ex：
    numbers = list(range(1,4))
    print(numbers)
out：
    [1,2,3]

数字列表的统计计算

min()、max()、sum() 的使用

列表解析

用于简化代码。

ex：
    numbers = [value**2 for value in range(1,4)]
    print(numbers)
out：
    [1,4,9]

切片（ python 术语，意为处理列表的部分元素）

其语法形式为：

    Array = ['gan','G','GA','GAN','GANR']
    print(Array[1:4])    //从第 1 个开始，到第 4 个结束（不包含第 4 个）。注意，从 0 开始计算。
out：
    ['G','GA','GAN']

如果没有指定开头索引值（如 Array[:4]）,则将从表头开始，其输出结果为：

['gan','G','GA','GAN']

类似的，没有指定末尾索引值（如 Array[1:]）,则从指定索引序号到列表末尾，其输出结果为：

['G','GA','GAN','GANR']

此外，索引序号可以为负数，请注意若为 [-3:]（负几都一样）,并不会给你打完一整个列表，而只会打印最后3个元素，直至列表结束。

切片的遍历

很简单，加个循环语句（如 for ）就行。

    ex：
        for cat in cats[:3]:
    out：
        [前面三个元素]

列表的复制

其实是对切片的特殊使用，形式为 [:]，意为从 0 开始，直至列表末尾（就是复制整个列表）。

ex：
    my_cars = friend_cars[:]
out：
    得到两个一毛一样的列表（但是互相独立）

注意不要这么写：

`my_cars = friend_cars`

这只是为 my_cars 另起一个变量名而已，这两个指代的还是同一个列表。

元组

元组的定义

列表中的元素可变，而元组不能变，其他相同。

存在的意义：在整个程序生命周期中保持一组元素值不变。（说白了就是常量）
其定义格式为：

    dimensions = (200, 50)        //用小括号，不再用中括号
    print(dimensions[0])
    print(dimensions[1])
out：
    200
    50

元组的遍历

ex：
    dimensions = (200, 50)
    for dimension in dimensions:
        print(dimension)
out：
    200
    50

修改元组的变量

虽然元组元素不可改变，但是可以给存储元组的变量赋值。

ex：
    dimensions = (200, 50)
    print("原来的：")
    for dimension in dimensions:
        print(dimension)
    dimensions = (400, 100)
    print("\n改变：")
    for dimension in dimensions:
        print(dimension)
out：
    原来的：
    200
    50

    改变：
    400
    100

注意：这里说的修改不是这么改：
dimensions[0] = 250
这是改不了的，元组存储的元素不可改变，只能从根源上改。

学习要点

  1. 列表的增、删、改、查
  2. for 的使用（和 C 有所不同，而且密切注意缩进的不同）
  3. 列表的复制
  4. 元组的应用

条件语句（不管怎么用，都不影响原值）

if 语句的使用

cars = ['audi', 'bmw', 'subaru', 'toyota']
for car in cars:
    if car == 'bmw':
        print(car.upper())
    elif car == 'audi':
        print(car.title() + " is very good!")                
    else:
        print(car.title())
out：

    BMW
    Subaru
    Toyota

常用的判断符号

== 判断是否相等
!= 判断是否不等
and 判断两条件是否同时为 True 。（相当于 && ）
or 判断是否有一个条件为 True 。（相当于 ||）
in 判断特定值是否已在列表中
    ex：
        Array = ['gan','GG']
        'gan' in Array
    out：
        True
not in 判断特定值是否不在列表中

学习要点

字典

字典的定义

类似于数据结构中的线性表，链表之类，具有将数据关联起来的作用。

字典的格式：

student_0 = {'name':'ganru','age':19} //由多个键值对构成

使用字典来存储用户提供的数据或在编写能自动生成大量键—值对的代码时，通常都需要先定义一个空字典。

字典的访问

ex：
    student_0 = {'name':'ganru','age':19}
    print(student_0['name'])
out：
    ganru

字典的增、删、改、查

增

ex：
    student_0 = {'name':'ganru','age':19}
    print(student_0)
    student_0['sex'] = 'man'
    student_0['graduated school'] = 'JYU'
    print(student_0)
out：
    {'name':'ganru','age':19}
    {'name':'ganru','age':19,'sex':'man','graduated school':'JYU'}

删

用 del 语句。

ex：
    student_0 = {'name':'ganru','age':19,'sex':'man','graduated school':'JYU'}
    print(student_0)
    del student_0['graduated school']
    print(student_0)
out：
    {'name':'ganru','age':19,'sex':'man','graduated school':'JYU'}
    {'name':'ganru','age':19,'sex':'man'}

注意：永删。

改

ex：
    student_0 = {'name':'ganru'}
    print("my name is" + student_0['name'] + ".")
    student_0['name'] = 'Gandalf'
    print("my name is" + student_0['name'] + ".")
out：
    my name is gan.
    my name is gandalf.

查

与 字典的访问 相似。

字典的遍历

遍历所有键值对

用 for 循环。

ex：
    student_0 = {
        'name':'ganru',
        'age':19,
        }
    for key,value in student_0.items()：    //声明两个变量，用于存储键值对中的键和值。
        print("\nKey: " + key)
        print("Value: " + value)
out：
    Key: name
    Value: ganru

    Key: age
    Value: 19

注意：遍历字典时，键值对的返回顺序与存储顺序不一定相同。

遍历无值键

用方法 keys() 来对字典进行遍历。

ex：
    student_0 = {
        'name':'ganru',
        'age':19,
        'sex':'man',
        'graduated school':'JYU',
        }
    for name in student_0.keys():        //其实可以不写 .keys()，输出依然不变（默认遍历）
        print(name.title())
out：
    name
    age
    graduated school

以某种顺序遍历字典中的所有键

用函数 sorted() 。

ex：
    favorite_languages = {
        'jen': 'python',
        'sarah': 'c',
        'edward': 'ruby',
        'phil': 'python',
        }

    for name in sorted(favorite_languages.keys()):
        print(name.title() + ", thank you for taking the poll.")
out：
    Edward, thank you for taking the poll.
    Jen, thank you for taking the poll.
    Phil, thank you for taking the poll.
    Sarah, thank you for taking the poll.

遍历所有值（无键）

用方法 values()。（可能得到很多重复项）

ex：
    favorite_languages = {
        'jen': 'python',
        'sarah': 'c',
        'edward': 'ruby',
        'phil': 'python',
        }

    for language in favorite_languages.values():
        print(language.title())
out：
    Python
    C
    Ruby
    Python

在 values() 的基础上用集合set()，可除掉重复项。

ex：
    favorite_languages = {
        'jen': 'python',
        'sarah': 'c',
        'edward': 'ruby',
        'phil': 'python',
        }

    for language in set(favorite_languages.values()):
        print(language.title())
out：
    Python
    C
    Ruby

字典的嵌套

比如 字典中的字典。

ex：
    alien_0 = {'color': 'green', 'points': 5}
    alien_1 = {'color': 'yellow', 'points': 10}
    alien_2 = {'color': 'red', 'points': 15}
    aliens = [alien_0, alien_1, alien_2]
    for alien in aliens:
        print(alien)
out：
    {'color': 'green', 'points': 5}
    {'color': 'yellow', 'points': 10}
    {'color': 'red', 'points': 15}

字典中存储列表

ex：
    student_0 = {
        'name': 'ganru',
        'language': ['python', 'C', 'C++'],
        }


    print(student_0['name'].title() + "'s favorite languages are:")
    for language in student_0['language']:
        print("\t" + language.title())

out：
    Ganru's favorite languages are:
        Python
        C
        C++

学习要点

  1. 如何定义字典，以及如何使用存储在字典中的信息；
  2. 如何访问和修改字典中的元素，以及如何遍历字典中的所有信息；
  3. 如何遍历字典中所有的键值对、所有的键和所有的值；
  4. 如何在列表中嵌套字典、在字典中嵌套列表以及在字典中嵌套字典。

Github+Hexo 搭建个人博客

发表于 2018-07-26

这个博客搭了我一整天，老实讲其实不难，只是我太蠢了，没有好好看人家写的文章。

写下这篇文章，旨在记录我搭建博客的过程，方便我下次想再复盘重新搭建一次博客时能少花点时间。如果你们有需要，也可参考参考，希望对你们有帮助。

OS：Windows 10

  • 1.环境准备

    • 1.1 Git
      这个第一个安装，直接去官网下载
    • 1.2 node.js
      一样，去官网下载好后一键傻瓜式下一步，当然，英语好的话或者如果你有其他的想法，Google、百度能帮你解决问题。
    • 1.3 hexo
      这个对于某些人可能有点困难（比如我=_=）,我们在后面表述。

  • 2.创建 GitHub 域名与 SSH 密钥部署

    • 2.1 注册 GitHub
      去官网注册，注意认真取 username，如果真心想好好玩的话，会方便很多。

    • 2.2 创建 Repository（存储库）

      写 Repository name 时尽量与 Owner 一致，对下面操作会方便一些。

    • 2.3 SSH 密钥部署
      在想放置的路径（我的是 D:\Hexo ）下右键打开 Git Bash Here ，输入：

      git config --global user.name "ganru"    //写你注册时的 username。
git config --global user.email "ganr1988@gmail.com"    //换成你自己的邮箱

      上面的步骤是为了配置你的 GitHub 姓名与邮箱，下面开始绑定密钥。

      输入代码：

      ssh-keygen -t rsa

      该带码表示指定 rsa 算法生成密钥，接着连续三个回车键（不需要输入密码） ，然后就会生成两个文件 id_rsa 和 id_rsa.pub ，而 id_rsa 是密钥，id_rsa.pub 就是公钥。win系统默认生成在 C:\Users\GR.ssh 下，注意文件是隐藏的。

      用记事本之类的软件打开 id_rsa.pub ，复制其中的内容。

      接下来如下图所示，进入SSH密钥页面

      title 不用写。将所复制的内容粘贴入内。

      此时返回 Git Bash Here ，输入代码验证下：

      ssh -T git@github.com

      至此密钥配对成功，以后就不用老是输入账户密码才能进 GitHub 进行终端操作了。

  • 3.hexo 的安装
    安装成功 Git 之后，我们在想放置的路径（我的是 D:\Hexo ）下右键打开 Git Bash Here ，输入：

    $ npm install -g hexo-cli

    这行代码表示安装 hexo ，验证是否成功可输入 hexo -v ,成功后如下图：

  • 4.在本地建设博客
    输入代码：

    hexo init

    该步骤是初始化的意思，会自动在目标文件夹（D:\Hexo）建立网站所需要的文件。

    npm install

    然后是安装 node_modules 到 D:\Hexo 目录中。这是建立 node 的模块，具体作用 Google。

    hexo server

    输入后，若出现这样的情况：

    说明 Hexo Server 已经启动了，在浏览器中打开 http://localhost:4000 ， 这时可以看到 Hexo 已为你生成了一篇 blog。（端口默认是4000）

    按下 Ctrl+C 停止 Server，接下来我们开始写一篇新的文章。

    可看到它的生成路径，我们可以使用一个支持 Markdown 语法的编辑器（比如 Sublime Text）尽情地编辑文件 My-New-Post.md 。

    写完文章之后，将 md（My-New-Post）文件生成静态网页：

    $ hexo generate

    该命令执行完后，会在 D:\Hexo\public\ 目录下生成一系列html，css等文件

  • 5.部署文件到 GitHub
    输入代码：

    $ hexo deploy

    可能会出错：

    这是因为我们没有安装 hexo-deployer-git，我们输入代码：

    npm install hexo-deployer-git –save

    在 D:\Hexo 目录下，可看到 _config.yml 文件，这个文件可以改博客主题、标题等等内容。现在我们直接找到并写成这段代码：

    deploy:
    type: git
    repository: git@github.com:ganru/ganru.github.io //记得改成你的 Repository name

    注意：使用 SSH git@github.com:ganru.github.io ，不要用 https，这个我也不明白为什么，网上很多人说弄 htpps，但我弄了好久没成功，这个倒成功了。（=_=）
    然后再改：

    注意：所有的冒号之后要空一格。如：type: git。

    执行代码：

    $ hexo deploy

    这时候就开始把本地的博客打包部署到你的 GitHub Repository （比如我的 ganru.github.io）里面了。

  • 6.通过网络登录博客
    打开浏览器，输入你的博客地址（https://ganru.github.io/），若正常显示，说明你的本地博客部署（上传）成功了。

  • 7.总结
    以后写博客就轻松了，我们只要在这基础上写文章就行：

    $ hexo new “......” //......表示文章的标题

    每次写完文章后，记得依次写这三个指令：

    hexo clean //该代码表示清除缓存
hexo generate （或者 hexo g）
hexo deploy     （或者 hexo d）

    文章才能上传到你的 GitHub 里面。

    还有，如果上面查看本地博客有问题的话，有很大可能是你装了福昕阅读器，因为它占用了 4000 这个端口，贼蛋疼。。
    附解决方法（如果你用得到的话）：

    $ hexo s -p 5000

    然后打开 http://localhost:5000/ 这个网址就行了。
