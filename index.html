Github+Hexo 搭建个人博客

这个博客搭了我一整天，老实讲其实不难，只是我太蠢了，没有好好看人家写的文章。

写下这篇文章，旨在记录我搭建博客的过程，方便我下次想再复盘重新搭建一次博客时能少花点时间。如果你们有需要，也可参考参考，希望对你们有帮助。

OS：Windows 10

  • 1.环境准备
    • 1.1 Git
      这个第一个安装，直接去官网下载
    • 1.2 node.js
      一样，去官网下载好后一键傻瓜式下一步，当然，英语好的话或者如果你有其他的想法，Google、百度能帮你解决问题。
    • 1.3 hexo
      这个对于某些人可能有点困难（比如我=_=）
      安装成功Git之后，我们在想放置的路径（我的是D:\Hexo）下右键打开 Git Bash Here，输入：
      $ npm install -g hexo-cli
      这行代码表示安装hexo，验证是否成功可输入 hexo -v
